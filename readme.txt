流程：
client  ---------------------   server
    |                              |
    |                              |
    |                              |
BpXmanService  -------------  BnXmanService
    |                           |
    |remote()->transact         | onTransact
    |                           |
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 此虚线以上是业务层，即需要用户来实现的代码
    |                           |
BpBinder     -------------    BBinder 
        |                     |
             binder driver
